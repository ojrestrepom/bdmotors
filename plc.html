Reflexiones

Rodrigo Deolan Vera

A nivel personal, este proyecto representó un desafío bastante grande. Sin embargo, el aprendizaje obtenido fue igual de significativo. Aunque ya tenía experiencia con softwares como Studio 5000 e Inventor, trabajar con Ignition, RS Linx y Siemens NX fue un reto completamente nuevo que me permitió ampliar considerablemente mis conocimientos, despertando aún más mi interés por el tema.

La posibilidad de lograr la interconexión entre estos softwares fue una experiencia muy gratificante, especialmente al ver en funcionamiento la supervisión de datos del gemelo digital. Además, todo el proceso de planeación, incluyendo el análisis de automatización para evaluar su viabilidad en planta, el estudio de costos, la implementación de análisis de riesgos y otras consideraciones clave para el desarrollo del proyecto, resultó ser una experiencia enriquecedora.

Lo único que considero como una dificultad fue el poco tiempo disponible para comprender a fondo los conceptos relacionados con el gemelo digital, SCADA y programación en Ladder. Aun así, para mí fue una experiencia muy valiosa y agradable poder participar en este proyecto.

Oscar Javier Restrepo

Para mí, este proyecto fue un gran reto. La gestión del tiempo y la organización de las actividades fueron aspectos fundamentales para poder llevarlo a cabo con éxito. A nivel personal, me enseñó a enfrentar problemas propios del trabajo en equipo y a tomar decisiones difíciles cuando fue necesario.

Desde el punto de vista técnico, comprendí que las consideraciones necesarias para implementar un sistema de automatización son mucho más amplias de lo que inicialmente imaginaba. No obstante, esto me permitió aprender y reconocer aspectos clave como el análisis de rentabilidad, el uso de indicadores como los KPIs y el OEE.

Además, abordar temas como la gestión de riesgos, el cumplimiento de normativas legales y el manejo de herramientas técnicas como Siemens NX, SCADA, programación Ladder e incluso el software de KUKA fue una experiencia que no solo fortaleció mis conocimientos, sino que también amplió mi perspectiva sobre todo el proceso de automatización.

En lo personal, aunque el tiempo disponible fue limitado, considero que asumir un reto como este vale completamente la pena. Tal vez reorganizaría el orden de los módulos para optimizar el aprendizaje dentro del tiempo asignado, pero en general, fue una experiencia que disfruté mucho.
